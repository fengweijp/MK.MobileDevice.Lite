MK MobileDevice Lite

MK MobileDevice Lite provides an interface for communicating with iOS devices developed by MK Industries and released under GPLv3. It can be used for various purposes, such as retrieving device info and saving and retrieving files from the device.

Using the library

Build the library from source with Visual Studio 2015, or grab the prebuilt binaries from Releases. Then just add an assembly reference to MK.MobileDevice.dll, and you're ready to go!
